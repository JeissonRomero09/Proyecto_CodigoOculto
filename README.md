# Código Secreto

Proyecto de desarrollo de un juego llamado **Código Secreto**, creado en **Java** utilizando **JavaFX** para la interfaz gráfica y siguiendo la arquitectura **MVC (Modelo - Vista - Controlador)**.

## Descripción
El juego consiste en que el jugador debe adivinar un código secreto generado por el sistema. Durante la partida el jugador tendrá intentos para descubrir el número correcto mientras el sistema valida las respuestas y gestiona el estado del juego.

## Tecnologías utilizadas

- JavaFX
- Scene Builder
- MySQL (para almacenamiento de datos en futuras funcionalidades)
- Git y GitHub
- Java 24
- JDBC
